# Растровый графический редактор с семантическим анализом

Это десктопный редактор для работы с растровыми изображениями. Проект выделяется тем, что умеет анализировать содержимое картинок. Кроме стандартных кистей, слоев и фильтров, здесь доступны инструменты, которые понимают смысл снимка: могут выделить объект по клику, удалить лишние детали с достройкой фона или найти нужную область по текстовому описанию. Вся обработка идет на устройстве пользователя, данные никуда не отправляются.

## Основные возможности

- Базовая обработка: цветокоррекция, морфологические операции, работа с каналами и контурами через OpenCV.
- Умное выделение: попиксельное обведение объекта просто по клику мыши (на базе MobileSAM).
- Удаление объектов: очистка изображения от лишних деталей с автоматической достройкой фона (модель LaMa).
- Поиск по тексту: выделение областей по описанию на русском языке (FastSAM + ruCLIP).
- Текстовое управление (в планах): выполнение команд вида "подними яркость и убери человека справа" через локальную языковую модель и Semantic Kernel.

## Технологии

- .NET 8 и Avalonia для интерфейса.
- SkiaSharp для отрисовки холста и слоев.
- OpenCVSharp для базовых операций с пикселями.
- ONNX Runtime для запуска нейросетей.
- Semantic Kernel для связки текстовых команд с кодом.
